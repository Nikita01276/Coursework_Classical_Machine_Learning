# Курсовая работа

## Прогнозирование эффективности химичеких соединений против вирусов

Данная работа посвящена применению методов машинного обучения для
прогнозирования эффективности химических соединений в качестве потенциальных
лекарственных препаратов против вирусов.

В качестве исходных данных используется датасет, содержащий информацию о 1000
химических соединениях с указанием их параметров эффективности (IC50, CC50, SI)


## Целевые параметры

IC50 — показатель того,при какой концентрации соединение подавляет 50% активности вируса.

CC50 — показатель того, при какой концентрации соединение убивает 50% клеток-хозяев.

SI  = CC50 / IC50.Чем выше SI, тем лучше потенциальный лекарственный кандидат.
В медицине порог SI > 8 рассматривается, как соотношение оптимальный порог


## EDA

6. Корреляционный анализ
7. Поиск мультиколлинеарности 
8. Поиск константных и почти константных признаков 

## Предобработка данных

Реализована в модуле utils.py:
1. Удаление служебного столбца-индекса
2. Удаление 33 константных признвков
3. Заполнение пропусков медианой каждого столбца
4. Удаление 33 признаков из пар с корреляцией > 0.95


## Сравнение моделей

Для задач регрессии применены 8 моделей:
LinearRegression 
Ridge,Lasso
DecisionTreeRegressor
RandomForestRegressor
GradientBoostingRegressor
SVR
KNeighborsRegressor

Для задач классификации применены 6 моделей:

RandomForestClassifier
GradientBoostingClassifier
SVC
KNeighborsClassifier


## Сравнение и выбор лучшей модели

1. Для регрессии используется метрика R^2 (коэффициент детерминации) на тестовой выборке.
2. Для классификации — F1-scor.
3. Также считаются: MAE, MSE, RMSE- для регрессии
4. Accuracy, Precision, Recall,ROC AUC -для классификации


## Выводы

1. Ансамблевые методы RandomForest, GradientBoosting показывают лучшие результаты в задачах регрессии — это объясняется способностью этих моделей улавливать сложные нелинейные зависимости
2. Линейные модели LinearRegression, Ridge, Lasso показывают  слабый результат, что говорит о наличии существенно нелинейных закономерностей в данных
3. SI — самая сложная для предсказания цель, т.к это производная величина (CC50/IC50), и её корреляции с отдельными признаками не превышают 0.16
4. Классификация работает лучше регрессии: даже когда модель не может точно предсказать значение, она может правильно отнести объект к классу выше или ниже порога
5. Для задачи SI > 8 наблюдается  падение качества — это связано с дисбалансом классов (35.7% / 64.3%)


## Рекомендации по улучшению результатов

1. Использовать продвинутые модели: XGBoost, LightGBM, CatBoost
   Они превосходят классический RandomForest на табличных данных.
2. Применить методы отбора признаков для удаления ненужных 
   признаков и снижения размерности.
3. Feature Engineering: создание новых признаков на основе доменных
   знаний
4. Увеличение размера датасета 
5. Балансировка классов для задачи SI > 8 через SMOTE
